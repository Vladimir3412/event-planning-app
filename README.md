# GroupPlan — сервис планирования групповых мероприятий

Курсовой проект. Интерактивный веб-сервис для совместного планирования мероприятий.

## Функциональность

- Создание и удаление мероприятий
- Управление участниками
- Задачи с назначением исполнителя
- Голосования с визуализацией результатов
- Учёт расходов с разбивкой на человека
- Смена статуса мероприятия
- Тёмная и светлая тема

## Стек

- Next.js 15, TypeScript
- Mantine UI, Tailwind CSS
- Архитектура FSD (Feature-Sliced Design)

## Запуск

```bash
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000)

## Структура

```
src/
├── app/          # страницы (Next.js App Router)
├── widgets/      # крупные блоки UI (layout, card-event, event-details)
└── shared/       # типы, константы
```
